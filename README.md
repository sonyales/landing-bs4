# Landing BS4

**Landing BS4** — адаптивный лендинг онлайн-курса по акварельной живописи «Петербург сквозь года». Страница содержит промо-блок, описание курса, список материалов, тарифы, FAQ и контактную информацию.
Demo: https://sonyales.github.io/landing-bs4/ 

Проект собран на **Vite**, стилизован с помощью **SCSS** и **Bootstrap 4**. Интерактивные элементы реализованы на **JavaScript**, **jQuery** и Bootstrap JS.

## Стек

* **HTML5**
* **SCSS**
* **Bootstrap 4**
* **JavaScript**
* **jQuery**
* **Vite**

## Функциональность

* адаптивная вёрстка лендинга;
* навигация по секциям страницы;
* плавный скролл к блоку с описанием курса;
* мобильное меню на базе Bootstrap Navbar;
* промо-блок с основной информацией о курсе;
* блок с описанием курса;
* блок с материалами для прохождения курса;
* раскрывающийся список красок;
* карточки тарифов;
* переключение стоимости тарифов между рублями и долларами;
* слайдер тарифов для компактного отображения карточек;
* FAQ-аккордеон на базе Bootstrap Collapse;
* футер с контактной информацией.

## Реализация

### Bootstrap 4 и SCSS

Bootstrap 4 используется для сетки, навигации, карточек, collapse-элементов и адаптивного поведения интерфейса. Поверх Bootstrap добавлены собственные SCSS-стили, переменные и миксины.

Основные визуальные параметры вынесены в SCSS-переменные: цвета, размеры, высота шапки, базовая типографика и параметры кнопок.

### Адаптивная вёрстка

Лендинг адаптирован под разные размеры экранов. Для изменения компоновки используются Bootstrap-классы и дополнительные media queries в SCSS.

Отдельно настроены:

* поведение навигации на мобильных экранах;
* размеры заголовков;
* отображение hero-изображения;
* карточки тарифов;
* раскрывающееся меню материалов.

### Интерактивные элементы

Интерактивность реализована в `src/js/main.js`.

В проекте настроены:

* скролл к следующей смысловой секции по кнопке «узнать подробности»;
* открытие и закрытие списка красок;
* переключение валюты внутри каждой карточки тарифа;
* обновление активного состояния кнопок валюты.

Для пересчёта цены используется фиксированный курс:

```js
const RATE = 70;
```

Стоимость хранится отдельно для каждой карточки, поэтому переключение валюты в одном тарифе не влияет на остальные.

### Сборка через Vite

Проект собирается через Vite. SCSS импортируется из JavaScript-точки входа, а Bootstrap JS подключается через `bootstrap.bundle.min.js`.

Для корректной публикации на GitHub Pages в `vite.config.js` указан base path:

```js
export default defineConfig({
  base: '/landing-bs4/',
});
```

## Запуск проекта

### Клонировать репозиторий

```bash
git clone https://github.com/sonyales/landing-bs4.git
cd landing-bs4/landing-bs4
```

### Установить зависимости

```bash
npm install
```

### Запустить проект в режиме разработки

```bash
npm run dev
```

После запуска Vite выведет локальный адрес приложения в терминале.

## Сборка проекта

```bash
npm run build
```

Готовая сборка будет создана в директории `dist`.

## Предпросмотр production-сборки

```bash
npm run preview
```

## Скрипты

| Команда           | Описание                                 |
| ----------------- | ---------------------------------------- |
| `npm run dev`     | Запуск проекта в режиме разработки       |
| `npm run build`   | Production-сборка проекта                |
| `npm run preview` | Локальный предпросмотр production-сборки |

## Деплой

В репозитории настроен workflow для деплоя на GitHub Pages. При пуше в ветку `main` GitHub Actions устанавливает зависимости, собирает проект и публикует содержимое директории `dist`.
